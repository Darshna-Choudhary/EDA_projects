# Data Analysis Projects

A collection of data analysis projects using Python, Pandas, NumPy, Matplotlib, and Seaborn.

## Projects
- Instagram Page Analytics – Engagement and growth analysis
- YouTube Trending Video Analysis – Trends in views, likes, comments, and audience engagement
- IMDb Movie Analysis – Movie ratings, votes, and genre trends

Each folder contains:
- Source code / notebooks
- Dataset
- Project-specific README
