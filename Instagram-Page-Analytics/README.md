# Instagram Page Analytics (EDA Project)

### Project Overview
This project performs exploratory data analysis (EDA) on Instagram page data to understand follower distribution, category-wise popularity, and the relationship between posting frequency and follower count. The goal is to derive meaningful insights from real-world social media data.


### Dataset
- Public Instagram influencer dataset
- Contains information such as:
    - Page name
    - Followers
    - Number of posts
    - Content category (Main topic)


### Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook


### Key Questions Explored
Which Instagram pages have the highest number of followers?


1.) How do average followers vary across different content categories?


2.) What is the overall distribution of followers on Instagram?


3.) Is there a relationship between the number of posts and followers?


4.) Does this relationship differ across content categories?


### Key Insights
- Instagram follower distribution is highly skewed, with a small number of pages holding a large share of followers.
- Content category plays a significant role in follower count; categories like Education, Sports, and Entertainment have higher average followers.
- There is no strong linear relationship between posting frequency and follower count.
- Category-wise analysis shows that Entertainment and Music pages can achieve high followers with fewer posts, while other categories show wide variation.
- Posting more content alone does not guarantee higher follower growth.


### Conclusion
The analysis highlights that Instagram popularity is influenced more by content domain, popularity, and audience interest than by posting frequency alone. Social media success follows a long-tail distribution, where only a few accounts dominate attention.
